# web3-crowdfund

Web3 application to create crowdfunding campaigns. The smart contract associated with this app is deployed to the Ethereum Rinkeby testnet. The client was built using React and Next.js.

## Tech Stack

-   JavaScript
-   Solidity
-   React
-   Next.js
-   Web3.js
-   Truffle
-   Mocha
-   Ganache
-   Semantic UI React
